# Arxiv Trend Data Feature Engineering: Documentation

이 문서는 `arxiv_trend_data.csv`를 기반으로 트렌드 관련 파생 변수(새 성분)를 생성하기 위해 `preprocess.py`에서 사용된 수학적 수식과 라이브러리 사용법을 정리한 문서입니다.

## 1. 파생 변수(새 성분) 설명 및 수식

### 1.1 카테고리 간 유사도 (Category Similarity)
두 카테고리가 시간이 지남에 따라 논문 출판 추세가 얼마나 비슷한지를 측정합니다. 이 스크립트에서는 **피어슨 상관계수 (Pearson Correlation Coefficient)** 를 사용했습니다.

*   **수식**:
    두 카테고리 $X, Y$의 출판량 시계열을 각각 $x_i, y_i$ (여기서 $i$는 날짜)라 할 때, 
    $$r_{xy} = \frac{\sum_{i=1}^{n} (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum_{i=1}^{n}(x_i - \bar{x})^2 \sum_{i=1}^{n}(y_i - \bar{y})^2}}$$
    (값이 1에 가까울수록 양의 상관관계가 강하고, -1에 가까울수록 음의 상관관계가 강함)

### 1.2 카테고리 간 공분산 (Category Covariance)
상관계수와 비슷하지만, 데이터의 척도(Scale)가 반영된 값입니다. 두 카테고리의 출판량이 함께 증가/감소하는 경향성을 보여줍니다.

*   **수식**:
    $$Cov(X, Y) = \frac{\sum_{i=1}^{n} (x_i - \bar{x})(y_i - \bar{y})}{n-1}$$

### 1.3 7일 이동평균 (7-Day Moving Average)
일일 출판량은 노이즈(주말/평일 차이 등)가 심할 수 있으므로, 과거 7일 동안의 평균을 계산하여 전반적인 단기 트렌드를 부드럽게(smoothing) 만듭니다.

*   **수식**:
    시간 $t$에서의 7일 이동평균 $MA_t$는 다음과 같습니다.
    $$MA_t = \frac{1}{7} \sum_{i=t-6}^{t} x_i$$
    (단, $x_i$는 $i$일의 특정 카테고리 논문 수)

### 1.4 트렌드 속도 (Trend Speed: 1차 미분)
트렌드의 변화량을 측정합니다. 여기서는 7일 이동평균이 하루 동안 얼마나 변했는지(변화율)를 1차 미분(Difference)으로 계산했습니다. 이 값이 양수이면 트렌드가 상승 중, 음수이면 하락 중임을 의미합니다.

*   **수식**:
    $$V_t = MA_t - MA_{t-1}$$

### 1.5 트렌드 가속도 (Trend Acceleration: 2차 미분)
속도가 변화하는 양을 측정합니다. 트렌드가 단순히 상승하는 것을 넘어 "점점 더 빠르게" 상승하는지(가속), "상승세가 둔화"되는지(감속)를 파악할 수 있습니다.

*   **수식**:
    $$A_t = V_t - V_{t-1} = (MA_t - MA_{t-1}) - (MA_{t-1} - MA_{t-2}) = MA_t - 2MA_{t-1} + MA_{t-2}$$

## 2. 사용된 Python 라이브러리 및 주요 함수

### 2.1 Pandas (`import pandas as pd`)
데이터 조작 및 시계열 분석을 위해 핵심적으로 사용되었습니다.

*   `df.explode('Category')`:
    *   **사용처**: 하나의 논문이 여러 카테고리를 가질 경우(`cs.IT math.IT`), 리스트로 분리한 뒤 각각을 개별 행으로 확장(Explode)하여 카테고리별 집계가 가능하게 합니다.
*   `df.groupby([...]).size()`:
    *   **사용처**: 날짜(`Date`)와 카테고리(`Category`)별로 묶어 일일 논문 출판 건수(Count)를 집계합니다.
*   `df.pivot(index='...', columns='...', values='...')`:
    *   **사용처**: 세로로 긴(long) 형태의 데이터를 가로로 넓은(wide) 시계열 매트릭스(행: 날짜, 열: 카테고리)로 변환합니다.
*   `df.reindex(pd.date_range(...), fill_value=0)`:
    *   **사용처**: 논문이 출판되지 않은 날짜(결측 날짜)를 0으로 채워 연속적인 시계열 데이터를 보장합니다.
*   `df.rolling(window=7, min_periods=1).mean()`:
    *   **사용처**: `window=7` 옵션으로 **7일 이동평균**을 자동으로 계산합니다.
*   `df.diff(periods=1)`:
    *   **사용처**: 시계열 데이터의 이산 미분(차분)을 수행합니다. 전일 데이터와의 차이를 구하여 **속도(1차 미분)** 및 **가속도(2차 미분)**를 산출합니다.
*   `df.cov()`, `df.corr()`:
    *   **사용처**: DataFrame에 포함된 모든 수치형 열(각 카테고리) 간의 **공분산 행렬**과 **피어슨 상관계수(유사도) 행렬**을 한 번에 계산합니다.
*   `pd.merge(...)`:
    *   **사용처**: 계산된 시계열 통계량(이동평균 등)과 유사도 파생 변수들을 원본 논문 데이터세트에 `Date`와 `Category` 기준으로 다시 병합합니다.

### 2.2 Numpy (`import numpy as np`)
수치 연산 보조를 위해 사용되었습니다.

*   `np.mean(...)`, `np.nan`:
    *   **사용처**: 논문이 여러 카테고리에 속할 때, 각 카테고리별로 산출된 값들의 평균(Average)을 논문의 대표값(새 성분)으로 구하거나, 계산 불가능한 경우 결측치(NaN)로 처리하기 위해 사용되었습니다.

### 2.3 Itertools (`import itertools`)
복잡한 반복문이나 조합을 생성할 때 쓰입니다.

*   `itertools.combinations(cats, 2)`:
    *   **사용처**: 하나의 논문이 3개의 카테고리(`A B C`)에 속할 경우, 두 개씩 짝지은 모든 조합(`(A,B), (B,C), (A,C)`)을 생성하여 카테고리 간 평균 유사도를 산출할 때 활용했습니다.
